# 解法：双指针

```cpp
int findLengthOfLCIS(vector<int>& nums)
{
    int n = nums.size();
    if (n == 1) return 1; // 数组长度为1时，最长连续递增子序列的长度为1
    int i = 0, j = 1, len = 0;
    while (j < n)
    {
        while (j < n && nums[j] > nums[j - 1]) ++j;
        len = max(len, j - i);
        i = j++;
    }
    len = max(len, j - i);

    return len;
}
```
