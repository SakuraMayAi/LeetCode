# 202. 快乐数

Problem Link: https://leetcode.cn/problems/happy-number/description/

## Solution 1

**Map**: Use an `Hash` whose type is `unordered_map` to store.

```cpp
unordered_map<int, int> Hash;
bool isHappy(int n)
{
    int qsum = n, tmp;
    while (Hash[qsum] == 0)
    {
        Hash[qsum] = 1;
        tmp = qsum, qsum = 0;
        do {
            qsum += (tmp % 10) * (tmp % 10);
            tmp /= 10;
        } while (tmp != 0);
        if (qsum == 1) return true;
    }
    
    return false;
}
```
