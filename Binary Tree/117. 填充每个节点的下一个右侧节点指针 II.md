# 解法一：迭代

该题层序遍历的迭代实现参考前置题，代码都不用改。

```cpp
Node* connect(Node* root)
{
    if (root == nullptr) return root;
    queue<Node*> qu;
    qu.push(root);
    while (!qu.empty())
    {
        int size = qu.size();
        for (int i = 0; i < size; ++i)
        {
            Node* node = qu.front();
            qu.pop();
            if (i != size - 1)
				node->next = qu.front();
			else node->next = nullptr;
            if (node->left) qu.push(node->left);
            if (node->right) qu.push(node->right);
        }
    }

    return root;
}
```
