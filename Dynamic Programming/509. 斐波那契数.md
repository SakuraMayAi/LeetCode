# 解法：动态规划

传统的递归处理时间复杂度太大了。考虑到斐波那契数列已经有了递推公式，可以用数组 `dp` 来保存递推过程中的关键数字。对于 `F(n)` 来说，`dp[0]` 保存前 `n - 1` 个数的和，`d[1]` 保存前 `n` 个数的和（斐波那契数列下标从0开始）所以每次循环都需要更新 `dp[0]` 和 `dp[1]`。不理解的读者可以手动推到一下递推的过程。

```cpp
int fib(int n)
{
    if (n <= 1) return n;
    int dp[2] = { 0, 1 };
    for (int i = 2; i <= n; ++i)
    {
        int sum = dp[0] + dp[1];
        dp[0] = dp[1];
        dp[1] = sum;
    }

    return dp[1];
}
```
